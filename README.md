# Розробка автоматизованої інформаційної системи для контролю особистих фінансів

Кваліфікаційна робота на підтвердження ступеня фахового молодшого бакалавра ([ВСП «ППФК НТУ «ХПІ»](http://polytechnic.poltava.ua)). 

Керівник роботи – **Кожем’якін Микола Віталійович**.

## Завдання до дипломної роботи

Розроблений програмний продукт має являти собою десктопний застосунок для автоматизованого контролю особистих фінансів. Програма повинна забезпечувати наступну функціональність: інтуїтивно зрозумілий графічний інтерфейс користувача, можливість безпечної авторизації за допомогою пароля, керування фінансовими операціями, налаштування користувацьких категорій, формування статистичних звітів з візуалізацією даних, локальне та безпечне збереження всієї інформації у вбудованій базі даних. 

## Використані технології

* Мова програмування C#
* Технологія WPF (Windows Presentation Foundation)
* База даних SQLite
* Entity Framework Core
